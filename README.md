# Carbon trinity
Rendering engine for the Carbon Game Engine

## 🛠️ Building

Build using provided `CMakeLists.txt` in the repository root.

For the Apple Silicon Metal build, standalone glTF/CMF samples, and the current
EVE scene-driver quality ladder, see the
[macOS Metal bring-up journal](docs/macos-metal-bringup.md).
The active dependency gates and evidence status are tracked separately in the
[EVE runtime-contract roadmap](docs/eve-runtime-contract-roadmap.md).
